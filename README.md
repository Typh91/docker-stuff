# Docker Tests

Continuous Integration environment:

* Jenkins
* Sonarqube
* Nexus Artifact Repository
* Git (in construction)
* Nginx

## How to use:

First of all, clone the repo:

```
git clone https://github.com/Typh91/docker-stuff.git
cd continuous-integration
```

Create the environment executing:

```
docker-compose up -d
```

Enjoy.
